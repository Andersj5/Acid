filter

filter takes an array or an object. If it is given an array, it will run a function on each item on the *array. If it is given an object, it will run a function on each key and or value of an object

filter
Returns
any: The value of each item in the given array after being run through a given function
Returns
any: The value of each property within an object after being ran through a given function
Example
Taking an array
const example = ['foo', 'bar', 'foobar'];
const fooFunction = () => {
 return example.length <4;
};
const value = filter(example, fooFunction);
//-> const value = ['foo', 'bar']
Taking an object
const example = [{
foo: bar,
bar: foo,
}];

 if hasValue(item.id) {
  return item.id

foo: bar,
